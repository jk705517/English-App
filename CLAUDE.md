# BiuBiu English 项目说明

## 项目简介
BiuBiu English 是一个面向中国女性（21-35岁）的英语学习平台，通过 YouTube vlog 视频配合字幕工具学习英语。

## 技术栈
- 前端：React + Tailwind CSS + Vite
- 主要文件：`src/pages/VideoDetail.jsx`（视频详情页，核心文件）
- 相关组件：`src/components/SubtitleItem.jsx`、`DictationInput.jsx`、`ClozeInput.jsx`、`IntensiveSentencePanel.jsx`
- 路由：React Router，视频页路由为 `/episode/:id`
- 本地开发：`npm run dev`（端口 5173）

## 当前字幕模式
双语 / 英文 / 中文 / 精读 / 听写 / 挖空 / 词卡（共7种）

## 播放控制条规范（任务6+7+8）

### 核心原则
**控制条按钮完全移出播放器画面，视频画面保持干净。**

### PC端控制条
位置：播放器画面正下方，独立一行。

按钮从左到右：
```
倍速 / 隐藏 / 全屏 | 上一句(←) / ▶播放 / 下一句(→) | A/B点 / 单句循环 / 间隔Xs / 单句暂停
```
- 每个按钮：图标 + 文字标签
- 上一句/播放/下一句：稍大圆形按钮
- 用竖线`|`分组，视觉上分3组

### 手机端控制条
位置：进入视频页时，**替换底部导航栏**（首页/收藏/本子/设置）；退出视频页时恢复导航栏。

按钮从左到右：
```
倍速 / 隐藏 | 上一句(←) / ▶播放 / 下一句(→) | A/B点 / 精听
```
- 上一句/播放/下一句：大圆形按钮居中
- 其余：小图标 + 文字标签

## 精听功能规范（任务7）

### 手机端
点击控制条「精听」按钮 → 从底部滑出设置面板，内容：
- **隐藏动态字幕**（toggle开关，默认关）
- **循环完当前句后暂停**（toggle开关，默认关）
- **循环间隔（秒）**：- 数字 + 按钮，默认3秒
- **循环次数**：- 数字 + 按钮，默认2次，最大值为∞（无限）
- 右上角「恢复默认值」按钮
- 所有设置实时生效，存入localStorage

### PC端
控制条直接显示独立按钮，不弹面板：
- **单句循环**：toggle按钮，开启后当前句播完自动重复
- **间隔Xs**：点击可调整秒数（-/+），显示当前值
- **单句暂停**：toggle按钮，开启后每句播完自动暂停

## A/B点片段循环规范（任务8）

A/B点合并为一个按钮，三段式交互：
1. **第一次点击**：设A点（记录当前时间戳），按钮高亮变色，显示「A●」
2. **第二次点击**：设B点，立即开始A→B片段循环，按钮显示「A↔B」
3. **第三次点击**：取消循环，按钮恢复默认样式

PC端和手机端都有此功能。
